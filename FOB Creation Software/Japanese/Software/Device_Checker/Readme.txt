【概要】
	DeviceChecker(以下本ソフトウエア)は、デバイスの状態を取得します。

【動作環境】
	Windows2000(SP3以降)、WindowsXP

【インストール】
1. 本ソフトウエアを、ハードディスク上にコピーしてください。
   
【アンインストール】
1. コピーしたファイルを全て削除してください。

【使用方法】
1. UDRW G2をパソコン本体USBポートに接続してください。
2. 本ソフトウエアを起動してください。
3. OutputFileボタンを押してください。
4. DeviceStatus.binが、実行ファイルと同じフォルダに出力されます。

【注意】
	本ソフトウエアで作成されたファイルは、
	お客様が使用することはできません。
