----------------------------------------《はじめに》--------------------------------------

　　UD RUN（以下、本ソフトウェアと記載）をご使用のパソコンにインストールすることにより、
　　Windows98SE及びWindowsMeのパソコンにて、UDRW G2（以下、本製品）内のROM領域にオートラン設
　　定されているファイルを、オートランさせる事が出来ます。

-----------------------------------------《注意事項》-------------------------------------

　　本ソフトウェアがインストールされていないWindows98SE、WindowsMeのパソコンでは、
　　本製品を接続してもオートランしません。

　　オートラン機能はかならず保証されるのではありません。ご使用になるパソコンの環境
　　によってオートランしない場合があります。

　　オートラン機能は、本製品を一旦パソコンから取り外し、再度接続してから有効となります。

　　本製品を接続したままパソコンで起動・再起動・スタンバイや休止状態からの復帰をした場合
　　はオートランしません。
　　※オートランを有効にするには、本製品を一旦はパソコンから取り外し、
　　再度接続し直してください。

-----------------------------------《インストール方法》-----------------------------------

	「UDRUN」フォルダにある「UDRUN.exe」を選択して、パソコンに本ソフトウェアを
	インストールします。



-----------------------------------《アンインストール方法》-------------------------------

	コントロールパネルのプログラムの追加と削除から削除をします。


------------------------------------《操作および設定など》--------------------------------

　　　　ソフトウェアの操作及び設定は必要有りません。
　　　　(ソフトウェアを実行した直後より有効となります）
