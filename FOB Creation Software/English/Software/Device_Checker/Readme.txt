DeviceChecker is used to output the state of the device. 

[System requirements]
	Windows2000(SP4),WindowsXP

[install]
1. Copy the DeviceChecker on the HDD. 

   
[uninstall]
1. Delete all copied files. 

[HowTo]
1. Connect UDRW G2 to the personal computer. 
2. Run the software.
3. Press OutputFile button.
4. DeviceStatus.bin is output to the same folder as the execution file. 

[note]
 You cannot use the file made with this software.
